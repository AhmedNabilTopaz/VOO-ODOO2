Odoo Keep initial address for delivery

A user-friendly application which manages the original/initial customer's shipping address in the delivery!

At odoo end, If the address of any customer is updated then it is changed in all the orders/deliveries generated previously for that customer. It is a huge issue in Odoo as it could lead to a lot of confusion and false reporting too.

In this case, if a customer requests for his previous order's receipt/shipping then the generated receipt/shipping will contain customer's updated address. In order to solve this issue, we have introduced our module "Odoo Keep initial address for delivery" which provides the feature to keep initial or original shipping address inside delivery generated for this customer earlier in Odoo. So whenever a customer asks for receipt/shipping of his previous order, then you can easily generate the receipt/shipping with the address which was originally used during the sale order.

Features:

  • Manage customer's initial addresses inside picking which were used during delivery.
  • Feature to update initial addresses of pickings in bulk, which don't have initial addresses.
  • Print customer's initial addresses inside picking/delivery receipt.
