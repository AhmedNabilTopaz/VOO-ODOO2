Odoo Keep initial address for orders

A user-friendly application which manages the original/initial customer's address in the sale order and invoice.

At odoo end, If the address of any customer is updated then it is changed in all the orders/invoices generated previously for that customer. It is a huge issue in Odoo as it could lead to a lot of confusion and false reporting too.

In this case, if a customer requests for his previous order's receipt/invoice then the generated receipt/invoice will contain customer's updated address. In order to solve this issue, we have introduced our module "Odoo Keep initial address for orders" which provides the feature to keep initial or original address inside sale order/ order invoices generated for this customer earlier in Odoo. So whenever a customer asks for receipt/invoice of his previous order, then you can easily generate the receipt/invoice with the address which was originally used during the sale order.

Features:

  • Manages customer's initial addresses inside sale order which were used during sale order.
  • Feature to update initial addresses of orders in bulk which don't have initial addresses.
  • Manages customer's initial addresses inside order invoice which were used during sale order.
  • Feature to update initial addresses of order invoices in bulk which don't have initial addresses.
  • Shows customer's initial addresses inside sale order receipt.
  • Shows customer's initial addresses inside order invoice receipt.
